Bit Blaster

Arduino interactive tester


RPN user interface is used to dynamically configure pins and exercise them.


Command format is arguments separated by commas followed by a key character.
If you forget to enter an argument many commands will infer a zero if that makes sense.

This program is insensitive to case, the UI library used is not.

Many commands configure pins, a few exercise them.
